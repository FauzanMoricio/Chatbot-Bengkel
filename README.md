# Project Title
**CHatbot Bengkel**

## Deskripsi
Chatbot Bengkel adalah sebuah website terkait bengkel untuk melihat informasi bengkel dan pembookingan service ditambahkan dengan chatbot untuk menanyakan informasi terkait bengkel dan melakukan pembookingan disana.

Dibuat Menggunakan:  
> HTML, CSS, PHP, MySql, JavaScript, Dialogflow.
